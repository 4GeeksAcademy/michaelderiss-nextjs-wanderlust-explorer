## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

# Design References

Below are 3 real interfaces that inspired the design for Wanderlust Explorer. These UIs feature clean discovery layouts, card-based browsing, and intuitive search/filter experiences:

1. [Airbnb Experiences](https://www.airbnb.com/s/experiences)  
   ![Airbnb Experiences Screenshot](https://a0.muscache.com/im/pictures/airbnbexperiences-hero.jpg)
   - Card grid, prominent search, filter bar, and beautiful imagery.

2. [Culture Trip](https://theculturetrip.com/europe/articles/the-best-experiences-in-europe/)  
   ![Culture Trip Screenshot](https://images.ctfassets.net/)
   - Editorial card layout, destination filters, and immersive visuals.

3. [GetYourGuide](https://www.getyourguide.com/)  
   ![GetYourGuide Screenshot](https://cdn.getyourguide.com/cdn-cgi/image/width=1920,height=1080,fit=cover/)
   - Top filter bar, search in URL, and modern card design.

_These references guide the visual and UX direction for the MVP explorer._
