# michaelderiss-nextjs-wanderlust-explorer

Wanderlust Explorer submission built with Next.js.

Project app lives in `nextjs-wanderlust-explorer/`.

## Run Locally

```bash
cd nextjs-wanderlust-explorer
npm install
npm run dev
```
